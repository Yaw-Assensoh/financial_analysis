#  Stock Market Analysis - SQL Portfolio Project

##  Project Description
A comprehensive stock market analysis using PostgreSQL to evaluate performance, risk, and investment opportunities across 6 major stocks from 2020-2025.

##  Key Features
- **10 Advanced SQL Analyses** - Performance, risk, correlation, and technical indicators
- **Interactive Dashboard** - Multi-page Streamlit application
- **Jupyter Notebook** - Complete visualization suite with business insights
- **Professional Documentation** - Methodology and business recommendations


##  Analytical Capabilities
- Performance benchmarking against market indices
- Risk assessment with volatility and Sharpe ratios
- Correlation analysis for portfolio diversification
- Technical trading signals and seasonal patterns
- Data quality validation and completeness checks

##  Business Questions Answered

1. **Which stocks outperformed the market?**
2. **What are the risk profiles of each stock?**
3. **Which stocks have the best risk-adjusted returns?**
4. **How correlated are these stocks for diversification?**
5. **What are the technical trading signals?**
6. **Seasonal and monthly performance patterns?**
7. **Extreme performance days analysis?**
8. **Quarterly business cycle performance?**
9. **Unusual trading volume detection?**
10. **Overall data quality validation?**

##  Technical Skills Demonstrated
- **Advanced SQL:** Window functions, CTEs, complex joins
- **Data Analysis:** Statistical functions, correlation analysis
- **Financial Analysis:** Sharpe ratio, volatility, moving averages
- **Data Management:** CSV import/export, data validation
- **Business Intelligence:** Converting data to actionable insights
- **Database:** PostgreSQL
- **Analysis:** SQL, Python, Pandas
- **Visualization:** Plotly, Matplotlib, Seaborn
- **Dashboard:** Streamlit
- **Tools:** pgAdmin
